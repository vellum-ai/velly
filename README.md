# velly

> **Warning**: This package is still under heavy beta testing and should not be run by others.

## Usage

```bash
bunx velly hatch
```

### Requirements

The following environment variables must be set:

- `GITHUB_APP_ID` – The GitHub App ID used to authenticate with the GitHub API
- `GITHUB_PRIVATE_KEY` – The GitHub App private key used to sign JWTs for token generation
